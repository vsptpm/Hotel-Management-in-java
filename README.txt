# Hotel-Management-in-java
Simple Hotel management system using Java and MySQL database
---------------------------------------------------------------------
SQL database 'hotelmanagement' contains following tables

---------------------ROOM--------------------------------------------
+--------+-------------+------+-----+---------+-------+
| Field  | Type        | Null | Key | Default | Extra |
+--------+-------------+------+-----+---------+-------+
| rno    | int(10)     | NO   | PRI |         |       |
| rtype  | varchar(50) | YES  |     | NULL    |       |
| rate   | int(20)     | YES  |     | NULL    |       |
| status | varchar(10) | YES  |     | NULL    |       |
+--------+-------------+------+-----+---------+-------+

---------------------CUSTOMER----------------------------------------
+---------+-------------+------+-----+---------+-------+
| Field   | Type        | Null | Key | Default | Extra |
+---------+-------------+------+-----+---------+-------+
| name    | varchar(10) | NO   |     |         |       |
| address | varchar(50) | NO   |     |         |       |
| idno    | int(20)     | NO   | PRI |         |       |
| roomno  | int(20)     | YES  | MUL | NULL    |       |
| days    | int(10)     | YES  |     | NULL    |       |
+---------+-------------+------+-----+---------+-------+

-----------------------USERS-------------------------------------------
+----------+-------------+------+-----+---------+-------+
| Field    | Type        | Null | Key | Default | Extra |
+----------+-------------+------+-----+---------+-------+
| username | varchar(50) | YES  |     | NULL    |       |
| password | varchar(50) | YES  |     | NULL    |       |
+----------+-------------+------+-----+---------+-------+

-----------------------ADMIN-------------------------------------------
+----------+-------------+------+-----+---------+-------+
| Field    | Type        | Null | Key | Default | Extra |
+----------+-------------+------+-----+---------+-------+
| username | varchar(20) | YES  |     | NULL    |       |
| password | varchar(20) | YES  |     | NULL    |       |
+----------+-------------+------+-----+---------+-------+
-----------------------------------------------------------------------
